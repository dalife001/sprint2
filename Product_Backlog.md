# Product Backlog

## Core Features
- User authentication (login / logout)
- Overview dashboard for spending and insights
- Add, edit, and delete transactions
- Categorize expenses (Food, Transport, Entertainment, Subscriptions)
- Search and filter transactions by name, category, or date

## Analytics & Insights
- Daily spending summary
- Monthly spending overview
- Compare current month spending with previous month
- Top expense categories with percentages
- Analytics charts for spending trends

## Banking & Integration
- Mock banking integration using TrueLayer Mock Bank
- Secure API communication between frontend and backend
- Transaction synchronization from mock bank

## AI & Smart Features
- Smart alerts for overspending in specific categories
- AI-based insights (or rule/token-based logic if AI is limited)

## Technical & Infrastructure
- API-first backend architecture
- Authentication and basic security mechanisms
- Performance optimization
- Responsive UI across devices
- Deployment and environment configuration
